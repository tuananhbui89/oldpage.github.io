Inverse Posenet

 

Motivation

Posenet [1] has proved the success of Deep Convolutional Neural Network in field of camera relocalization to predict camera pose (position and orientation) from a image. This work has demonstrated that convnets can be used to solve complicated out of image plane regression problems. However, in the inverse problem, such that regression (inference) a visual sence from a pose where we desire to view with specific angle still a hard problem (even with human). One of problem is that we donâ€™t have a good measure method to evaluate the generated sence compare with the ground truth image. Another problem is that the ability of model inference the full information of a object taken from several images is still unknown this time. Fortunately, with the development of Generative Adversarial Network, we now have a more efficient method to evaluate a generated model. We propose a Inverse Posenet, the end2end system which can generate (inference) a visual sence from location you want

Architecture

Overall system

We use a simple system in which the Generator attempts to deceive the Discriminator by producing a fake sence driectly from the 6 DoF vector. The input of Discriminator is a set consisting of a Pose vector and a corresponding sense (real or fake image).

Generator

Follow the Pix2Pix paper [2], instead of using a prior distribution as input as in original GAN, we use a Dropout in some layers as a gaussian prior distribution.

Discriminator

As in Conditional GAN [3] we use the 6 DoF vector as a “label” for each corresponding sence, thus, we add them to the second last layer.

Detail

Generator

Layer Func Input Output Sub-process
G_d1 Linear 7x1 2048x1
Reshape 2048x1 2x2x512
G_d2 Deconv 2x2x512 4x4x512 Relu + BN + Dropout
G_d3 Deconv 4x4x512 8x8x512 Relu + BN + Dropout
G_d4 Deconv 8x8x512 16x16x512 Relu + BN
G_d5 Deconv 16x16x512 32x32x256 Relu + BN
G_d6 Deconv 32x32x256 64x64x128 Relu + BN
G_d7 Deconv 64x64x128 128x128x64 Relu + BN
G_d8 Deconv 128x128x64 256x256x3 Relu

Discriminator

Layer Func Input Output Sub-process
D_0 Conv 256x256x3 128x128x64 Lrelu
D_1 Conv 128x128x64 64x64x128 Lrelu + BN
D_2 Conv 64x64x128 32x32x256 Lrelu + BN
D_3 Conv 32x32x256 16x16x512 Lrelu + BN
Reshape 16x16x512 131072x1
D_4 Linear 131072x1 7x1
Concat 7x1 + 7x1 14x1
D_5 Linear 14x1 1
